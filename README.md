# Atom Format Parser

This project is to create my own parser for the Atom format as defined in [RFC4287](https://www.ietf.org/rfc/rfc4287.txt).

**This is not designed to be performant or complete. This is just cause I can.**

## Building
This project uses [UV](https://github.com/astral-sh/uv). It has a [main.py](./main.py) file for convenience, so you can just run via `uv run ./main.py`
